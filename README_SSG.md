# NeuroDeny - Static Site Generator + CMS

Профессиональный сайт неврологической клиники на базе **11ty (Eleventy)** + **Decap CMS**.

## 🚀 Технологии

- **SSG:** 11ty (Eleventy) - быстрый и гибкий генератор статических сайтов
- **CMS:** Decap CMS - Git-based система управления контентом
- **Шаблоны:** Nunjucks
- **Стили:** CSS + Tailwind CSS (CDN)
- **Деплой:** GitHub Actions → GitHub Pages

## 📁 Структура проекта

```
src/
├── _data/              # Данные (JSON)
│   ├── settings.json   # Настройки сайта, контакты
│   ├── doctors.json    # Врачи
│   ├── services.json   # Услуги и цены
│   └── reviews.json    # Отзывы
├── _includes/          # Компоненты (navigation, footer, mascot)
├── _layouts/           # Шаблоны страниц
│   ├── base.njk        # Базовый layout с SEO
│   └── blog-post.njk   # Layout для постов блога
├── admin/              # Decap CMS админка
│   ├── config.yml      # Конфигурация CMS
│   └── index.html      # UI админки
├── blog/               # Статьи блога (Markdown)
└── index.njk           # Главная страница

css/                    # Стили (из предыдущих лаб)
images/                 # Изображения
.eleventy.js            # Конфигурация 11ty
```

## 🛠 Локальная разработка

### Установка

```bash
npm install
```

### Запуск dev-сервера

```bash
npm start
```

Сайт будет доступен на `http://localhost:8080`

### Сборка для продакшена

```bash
npm run build
```

Результат в папке `_site/`

## 📝 Управление контентом через CMS

### Доступ к админке

После деплоя откройте: `https://nickseen.github.io/NeuroDeny/admin/`

### Что можно редактировать:

✅ **Настройки сайта** - SEO, контакты, график работы, соцсети  
✅ **Врачи** - добавлять/редактировать специалистов, фото, опыт  
✅ **Услуги** - обновлять прайс-лист  
✅ **Отзывы** - добавлять отзывы пациентов  
✅ **Блог** - создавать полезные статьи (SEO!)  

### Workflow:

1. Открываете `/admin/`
2. Авторизуетесь через GitHub
3. Редактируете контент в визуальном редакторе
4. Сохраняете → автоматически создается commit
5. GitHub Actions пересобирает сайт
6. Обновления на сайте за 1-2 минуты

## 🔐 Настройка CMS (первый раз)

### 1. Включить Netlify Identity (для авторизации)

Есть 2 варианта:

**Вариант A: Netlify (рекомендуется)**
1. Зарегистрируйтесь на [netlify.com](https://netlify.com)
2. Подключите репозиторий
3. Включите Identity в настройках
4. Включите Git Gateway

**Вариант B: GitHub OAuth (альтернатива)**
Настроить OAuth приложение в GitHub Settings.

### 2. Настройки GitHub Pages

В репозитории Settings → Pages:
- Source: GitHub Actions
- Branch: не важно (используется Actions)

## 🚀 Деплой

### Автоматический деплой

При каждом push в ветку `lab4`:
1. GitHub Actions запускается
2. Устанавливаются зависимости
3. Собирается сайт (`npm run build`)
4. Публикуется на GitHub Pages

### Ручной деплой

```bash
npm run build
# Залить _site/ на хостинг
```

## 📝 Добавление новой статьи в блог

### Через CMS (просто):
1. Открыть `/admin/`
2. Blog → New post
3. Написать статью в редакторе
4. Сохранить

### Вручную (для разработчиков):
Создать файл в `src/blog/`:

```markdown
---
layout: blog-post.njk
title: "Заголовок статьи"
description: "Описание"
date: 2026-03-10
author: "Имя автора"
image: "/images/cover.jpg"
tags: ["неврология", "здоровье"]
---

Текст статьи...
```
